# Quiz System

## 📖 Overview

Quiz System is a simple Java console-based application that allows users to answer multiple-choice questions and receive an instant score with performance feedback.

This project is suitable for Java beginners to understand arrays, loops, conditional statements, and user input using the Scanner class.

---

## 🚀 Features

- Multiple-choice questions
- Automatic answer validation
- Instant score calculation
- Percentage calculation
- Performance grading
- Simple console interface

---

## 🛠 Technologies Used

- Java
- JDK 8 or above
- VS Code / IntelliJ IDEA / Eclipse
